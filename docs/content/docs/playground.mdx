## How to Use

1. **Configure Props** - Use the left panel to adjust settings
2. **Test Import** - Click "Open CSV Importer" to try it live
3. **Sample Data** - Download or copy sample CSV files for testing
4. **Copy Code** - Get the generated code for your configuration

## Features You Can Test

- **Appearance**: Dark mode, primary color, language
- **Columns**: Add/remove columns, set types, add validators
- **Behavior**: Modal vs embedded, header selection
- **Validation**: Required, unique, min/max, patterns
- **Data**: Test with clean data, errors, or edge cases
